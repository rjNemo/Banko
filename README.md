# Banking App


## Features

### Users
* Admin or Customer
* List of Accounts

### Accounts
* Balance
* CreditLimit
* User
* List of Operations

### Operations
* Date
* Account
* Amount
* Description
* Type


## Stack

### Front-end
* React for client-side code

### Back-end
* ASP.NET Core 3.1 and C# for cross-platform server-side code
* PostgreSQL Relational Database
